Install Karne Ke Aasaan Qadam

In steps ko follow karein aur seconds mein Shahkot Directory set up ho jaayegi

1

📲 Website Ya Link Browser Mein Kholo

Apne mobile ke browser (Chrome ya Safari) mein Shahkot Directory ka link open karein. Direct search kar sakte hain ya diya gaya link use karein.

💡 Chrome browser use karna best hai
2

📤 Share Ya 3-Dots Icon Dabayein

iOS (iPhone): Browser mein Share button (□↑) dabayein. Android: Top-right corner mein teen dots (⋮) ya menu icon par click karein.

💡 iPhone par Safari se karna best hai
3

🏠 "Add to Home Screen" Select Karein

Menu mein "Add to Home Screen" ya "Install App" ka option milega. Is par tap karein — app bilkul native app ki tarah install ho jaayegi.

4

✓ Naam Confirm Karke "Add" Dabayein

"Shahkot Directory" naam dikhega — isko confirm karein aur Add dabayein. App aapke home screen par aa jaayegi bilkul normal app ki tarah.

🎉 App install ho gayi! Ab home screen se directly open karein
5

💻 PC / Desktop Par Install Karna

Chrome mein website open karein → address bar ke daayein taraf Install icon milega → click karein → "Install" dabayein. Desktop shortcut bhi ban jaayega.

💡 Yeh PWA hai — bilkul desktop software ki tarah kaam karta hai

🎭 Search Filters

Filter Ka Istemal Karein

Search ko aur bhi accurate banaayein — bilkul woh dhundhein jo chahiye

🔠

Shop Name Filter

Sirf shop ke naam se search karein. Agar aapko ek specific dukaan ka naam pata hai toh yeh sabse fast option hai.

🔠 Example: "Shop Name" select karke "Hassan" type karein → Sirf "Hassan" naam wali dukaanein dikhein gi
👤

Owner Filter

Dukandaar ke naam se search karein. Agar dukan ky bary main kuch nahi pata or sirf owner ka name pata hy to sirf (owner filter) istemal kerain.

👤 Example: "Owner" filter par "Aslam" type karein → Aslam sahib ki saari dukaanein list ho jaayengi
🏷

Category Filter

Business type ke mutabiq search karein. Medical, Electronics, Clothing — sirf woh category type karein aur filtered results paayein.

🏥 Example: "Category" select karke "Medical" type karein → Sirf medical shops list ho jaayengi
🛣

Road / Area Filter

Kisi specific gali, road ya mohalle ki dukaanein dhundhin. Apne qareeb ki shops dekhna chahte hain toh yeh filter best hai.

📍 Example: "Road" filter par "Main Bazaar" type karein → Main Bazaar ki saari dukaanein mil jaayengi

❓ FAQ

Aksar Pooche Jaane Wale Sawal

Koi confusion ho toh yahan zaroor jawab milega

Kya Shahkot Directory bilkul free hai?▾
Haan, Shahkot Directory 100% free hai. Search karein, shop ki details dekhein, call karein, WhatsApp karein — sab kuch bilkul muft. Koi hidden charges nahi hain.
Agar dukaan band ho jaaye toh information galat nahi ho jaati?▾
Business owners hamari team ko changes share karte hain — aur team foran directory update kar deti hai. Isliye information hamesha latest rehti hai. Verified badge wali shops specially monitored hain.
Search karne par koi result nahi aa raha — kya karein?▾
Chand cheezein try karein: (1) Spelling check karein, (2) Filter "All" par set karein, (3) Chhota ya simpler keyword use karein, (4) Category chip se browse karein — direct search se zyada results milte hain. Agar phir bhi nahi mila toh woh dukaan abhi registered nahi hai.
Apni dukaan Shahkot Directory mein add karwana chahein toh?▾
Bilkul! Agar aap Shahkot main kisi bhi business ky owner/manager hain toh apni shop list karwa sakte hain. Yeh ek affordable monthly subscription par available hai. Humse directly WhatsApp ya contact page se rabta karein — hamari team aapki poori madad karegi.
Google Maps button tap kiya par map nahi khula — kya masla hai?▾
Google Maps app aapke phone par installed honi chahiye. Agar nahi hai toh browser mein Google Maps khulega. Agar woh bhi nahi khula toh internet connection check karein. Kabhi kabhi pehli baar permission maangta hai — "Allow" dabayein.
"Verified" badge kya hai or jin par nahi hy wo shops?▾
Verified badge wali shops ko hamri team zyada monitored karti hai. Is main owner daily updates karwa sakta hai jis sy latest information ap tak pohanchti hai.

Jin par verified ka badge nahi hy unki bhi details bilkul correct hoti hain magar yeh monthly bases par update hoti hain.
Free Tip!▾
Ap jo bhi dukan, busniess, resturent etc search karain to jany sy pehly us busniess ky contact/whatsapp number par lazmi rabta karain. Takee ap ghar bethy hi apny kaam ky mutalik us busniess ki kuch details ly lain. Or apka fazool main chakkar na lagy.
